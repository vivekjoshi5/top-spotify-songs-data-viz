About Dataset 📃

This dataset contains information about top 2000 tracks on Spotify from 2000-2019. The data contains 18 columns each describing the track and its qualities including the popularity, year of release, genre and artists.

Methodology 🧰

I will Excel spreadsheet program to clean the data & Tableau for analyzing the data and creating the final dashboard.

Data Cleaning 🧹

There are no null values in the dataset. There were 59 duplicate values which have been removed. The dataset contains multiple songs with similar title. To identify them I have created a separate column naming it as “id” that has a unique value for each song.
